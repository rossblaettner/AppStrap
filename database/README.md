
# Database Layer

PostgreSQL provides:

- relational persistence
- graph metadata
- event sourcing
- inheritance tracking
- NTFS-style ACLs

---

## Benefits

### Single Source of Truth

All major system concepts remain:

- transactionally consistent
- replayable
- queryable

### JSON + Relational Hybrid

Supports:

- strict schemas
- evolving structures
- dynamic templates

### Stored Procedures

Complex logic handled server-side:

- effective permissions
- inheritance resolution
- replay traversal
- rollback planning
